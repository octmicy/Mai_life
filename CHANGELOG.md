# 更新日志

## [1.5.0] - 2026-07-13

### 新增

- 私人/公开书柜、文档修订和创作运行记录。
- 生活、梦境、日记灵感到提纲、正文、审校、修订、归档的完整流水线。
- 小说片段、诗、随笔、短剧、分镜、角色和世界观七类作品。
- 可选外部阅读插件 API、私人第一人称批注和二进制忽略规则。
- 书柜查询、文本阅读和管理员立即创作命令。
- 创作提纲、正文、审校与阅读批注的独立模型任务路由。

### 隐私与存储

- 创作默认关闭，未确认 SQLite 明文存储风险时不会运行。
- 私人来源不能提升为公开；朋友的查询在 SQLite 层只返回公开作品。
- 已有抽象日记在 schema 升级时回填到私人书柜，不复制聊天记录。
- 配置 Schema 升级为 `1.5.0`，SQLite Schema 升级为 v6。

## [1.4.0] - 2026-07-13

### 新增

- 白名单群聊的独立话题缓冲、匿名短摘要和成员活跃时间。
- 按唯一群别名和群友关系词条执行 `/mai_relay` 显式转述。
- 通过 Host 标准 `at` 消息段实现 SnowLuma/NapCat 一致的首段真实 @。
- 公开群话题到私聊的低频定向主动契机，遵守用户授权、离群时长和主动额度。
- 群聊判断与匿名转述摘要的独立模型任务路由。

### 安全与兼容

- 社交功能默认关闭，只处理显式群白名单；不保存群聊原文。
- Planner 沉默或发送失败不记为已发送，歧义群名和群友别名直接拒绝。
- SnowLuma 与 NapCat 共用 Host 消息契约，不引入适配器专属配置或单端功能。
- 配置 Schema 升级为 `1.4.0`，SQLite Schema 升级为 v5。

## [1.3.0] - 2026-07-13

### 新增

- RSS/Atom 新闻源、可读正文提取和可选 B 站插件 API 来源。
- 自建 SearXNG 与通用 JSON 主动搜索连接器。
- 新闻、搜索结果与麦麦人格、状态、能力和日程的自我关联。
- 近期新闻见闻与探索笔记命令，以及新闻/搜索/关联模型任务路由。

### 安全与兼容

- 所有联网能力默认关闭，不内置公共搜索节点或必须下载的依赖。
- 使用 Python 标准库 HTTP/XML/HTML，兼容 Windows/Linux 并遵循系统代理。
- 请求失败持久化退避，正文不可读时只保留来源摘要，不编造内容。
- 私聊话题默认不进入外部搜索词；搜索词本地清理账号、URL、邮箱与 @ 对象。
- 外部内容以不可信背景注入 Prompt 和 Planner，不执行来源中的指令。
- 配置 Schema 升级为 `1.3.0`，SQLite Schema 升级为 v4。

## [1.2.0] - 2026-07-13

### 新增

- 梦境摘要与最多 5 条醒后碎片，梦境余韵可形成低频主动契机。
- 每日抽象日记；只使用麦麦生活节点、梦境和互动数量，不复制聊天原文。
- 重要日期自动识别、模糊日期候选、手动确认与目标用户专属提醒。
- 基于日程实践证据的技能成长、每日增长上限和熟悉度阶段。
- 日记、日期、技能命令及对应模型任务路由。

### 调整

- 配置 Schema 升级为 `1.2.0`，SQLite Schema 升级为 v3。
- 重要日期以匿名提示影响全局日程，避免共享生活线泄露朋友身份。
- 删除重要日期时同步撤销尚未消费的日期主动契机。

## [1.1.0] - 2026-07-12

### 新增

- 配置私聊的文本、单图与合并转发消息收口。
- 同一消息锚点的 Replyer 轮次锁，抑制重复回复。
- 主人/朋友关系角色与用户级主动额度。
- 疑难图片、引用图片、转发图片和 GIF 视觉摘要。
- 工作日、农历节气、平台及消息媒介环境快照。
- 快速/推理/创作/视觉任务编排与 Token 统计。
- 轻量当前意图和未完话题元数据。
- SnowLuma 与 NapCat 入站消息归一化及双适配器契约测试。

### 调整

- 休息闸门增加夜间/午休双窗口，并改为发送成功后才提交醒来状态。
- 主动消息额度改为平台实际发送成功后计数。
- 模型任务名由无效的 `reply` 修正为 `replyer`，视觉任务使用 `vlm`。
- 配置 Schema 升级为 `1.1.0`，SQLite Schema 升级为 v2。

### 暂缓

- 当前适配器不转发 OneBot 撤回通知，因此撤回取消未在本版本实现。

本项目从原 `Mai_love` 的重构实验中独立出来，自此使用独立仓库、插件 ID 和版本序列。

## [1.0.0] - 2026-07-12

### 新增

- 全局共享的麦麦生活状态：精力、饥饿、心情、健康、睡眠和位置感。
- 基于城市名的 Open-Meteo 天气背景，无需用户配置经纬度。
- 每日生活框架与临近场景细化。
- 夜间睡眠、午休、醒来缓冲和梦境余韵。
- 多用户独立关系温度、活跃度、免打扰和主动消息额度。
- 规则筛选与 MaiSaka Planner 终审结合的主动私聊。
- Planner/Replyer 分层生活上下文。
- SQLite 事务存储、配置热更新和可取消后台任务。
- WebUI 中文配置标签、说明与英文 i18n。
- 用户命令、公共 API、测试和使用文档。

### 项目标识

- 插件目录：`Mai_life`
- 插件 ID：`maibot-community.mai-life`
- 展示名称：`麦麦生活`
- 初始独立版本：`1.0.0`
- 配置 Schema：`1.0.2`

- 不自动迁移或删除旧 `Mai_love` 运行数据。
