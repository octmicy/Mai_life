# 更新日志

## [1.1.0] - 2026-07-12

### 新增

- 配置私聊的文本、单图与合并转发消息收口。
- 同一消息锚点的 Replyer 轮次锁，抑制重复回复。
- 主人/朋友关系角色与用户级主动额度。
- 疑难图片、引用图片、转发图片和 GIF 视觉摘要。
- 工作日、农历节气、平台及消息媒介环境快照。
- 快速/推理/创作/视觉任务编排与 Token 统计。
- 轻量当前意图和未完话题元数据。
- SnowLuma 与 NapCat 入站消息归一化及双适配器契约测试。

### 调整

- 休息闸门增加夜间/午休双窗口，并改为发送成功后才提交醒来状态。
- 主动消息额度改为平台实际发送成功后计数。
- 模型任务名由无效的 `reply` 修正为 `replyer`，视觉任务使用 `vlm`。
- 配置 Schema 升级为 `1.1.0`，SQLite Schema 升级为 v2。

### 暂缓

- 当前适配器不转发 OneBot 撤回通知，因此撤回取消未在本版本实现。

本项目从原 `Mai_love` 的重构实验中独立出来，自此使用独立仓库、插件 ID 和版本序列。

## [1.0.0] - 2026-07-12

### 新增

- 全局共享的麦麦生活状态：精力、饥饿、心情、健康、睡眠和位置感。
- 基于城市名的 Open-Meteo 天气背景，无需用户配置经纬度。
- 每日生活框架与临近场景细化。
- 夜间睡眠、午休、醒来缓冲和梦境余韵。
- 多用户独立关系温度、活跃度、免打扰和主动消息额度。
- 规则筛选与 MaiSaka Planner 终审结合的主动私聊。
- Planner/Replyer 分层生活上下文。
- SQLite 事务存储、配置热更新和可取消后台任务。
- WebUI 中文配置标签、说明与英文 i18n。
- 用户命令、公共 API、测试和使用文档。

### 项目标识

- 插件目录：`Mai_life`
- 插件 ID：`maibot-community.mai-life`
- 展示名称：`麦麦生活`
- 初始独立版本：`1.0.0`
- 配置 Schema：`1.0.2`

### 说明

- 不继承 `Mai_love` 的发布版本号。
- 不自动迁移或删除旧 `Mai_love` 运行数据。
- 后续更新在本仓库中独立维护。
